# HEAL Wellness Website

## Run locally

1. Open the folder in VS Code.
2. Open the terminal.
3. Run `npm install`.
4. Run `npm run dev`.
5. Open `http://localhost:3000`.

## Main files

- `app/page.tsx` — homepage content and sections
- `app/globals.css` — colors, layout, animation and responsive styling
- `public/images` — HEAL logo, team photos and office photo
